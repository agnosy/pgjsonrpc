pgjsonrpc
=========

Synopsis
--------

  An extension to support the JSON-RPC 2.0 specification.

Description
-----------

Implementation of JSON-RPC 2.0 specification as PostgreSQL extension.

Usage
-----

  CREATE EXTENSION pgjsonrpc;
  SELECT jsonrpc.execute('{"id": 1, "method": "test"}');
